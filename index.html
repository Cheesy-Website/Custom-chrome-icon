INSTALL THIS AS A SHORTCUT

This is for Chrome OS only, click the 3 dots in the top right, then hover over "Cast, save, and share", then click on "Create shortcut..." and set the name you want there, also make sure that open in window is unchecked.

To add a custom icon, just fork this repo, and either rename the favicon.ico file to something else and rename one of the other files to favicon.ico, or you can import your own favicon.

You can fork the repo here.
